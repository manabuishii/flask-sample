# flask-sample
Flask sample

# Install flask

Install Flask (0.12)

# Run

```
python flasksample.py
```

it works well

```
$ python flasksample.py
 * Running on http://0.0.0.0:15000/ (Press CTRL+C to quit)
```

```
docker run --rm manabuishii/flask-sample:0.1.0
```
